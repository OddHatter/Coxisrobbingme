About This Project


Once I have finished commenting the code, I will provide a link to the github page where I will be uploading the source code.

To give a few details regarding this project, the device being used to test speeds is a Jetson Nano 2GB with a 1Gbps ethernet port, that is connected directly to the modem using a cat 6 cable.

The nano runs a script every 20 mintues that uses speedtest-cli to get speedtest data and insert it into a database.
Another script is run on the server every 10 minutes, to query the database for download speeds, and calculate the average difference in package speed:actual speed,
and use that data to calculate that difference in terms of how much I pay vs what service I get.

This is a work in progress and will continue to improve my code and hardware to factor in upload speed as well.
